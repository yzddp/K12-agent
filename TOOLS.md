# TOOLS.md — 学伴工具指南

## 核心能力

### 学生辅导
- `read` — 读取错题图片 / 题目文本 / 作文文本
- 苏格拉底式引导（内置人格，无需额外工具）
- 多轮对话 session 管理
- 作文批改（语文/英语结构化评估）

### 家长服务
- 初始化配置（录入孩子信息、课表、学期日历、积分规则）
- 结构化数据处理（知识标签、错误分类）
- 周报生成（周一 08:00 自动推送）
- 学情趋势分析（进步/退步判断）
- 积分报告与兑换管理

### 数据管理
- 错题库读写（error_questions）
- 知识点掌握度更新（knowledge_mastery）
- 学习计划调整（study_plans）
- 获奖记录（awards）
- 学期日历（semester_calendar）
- 积分账本（points_transactions，只追加）

### 数据来源推荐
- 高考志愿/选科：教育部阳光高考（gaokao.chsi.com.cn）、各省教育考试院官网

## 输出风格

- 孩子模式（群聊@）：纯对话，可含引导性提问
- 家长模式（私聊）：纯文本 + 【】小标题
- 所有输出简洁直接，不废话
